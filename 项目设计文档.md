# 校园职通 - 大学生兼职平台

## 项目概述

### 项目背景
"校园职通"是一个专为大学生群体设计的兼职招聘平台，旨在连接企业雇主和在校大学生，提供安全、可靠、高效的兼职信息服务。随着大学生兼职需求的日益增长，传统兼职渠道存在信息不对称、安全隐患等问题，需要一个专业化的平台来解决这些痛点。

### 项目愿景
成为全国领先的大学生兼职服务平台，构建安全、便捷、可信赖的校园人才生态系统。

### 核心价值
- **对学生**：安全可靠的兼职机会，便捷的求职体验
- **对企业**：高质量的大学生人才，高效的招聘流程
- **对平台**：良性的发展生态，可持续的商业模式

- **多方式登录**：邮箱/手机号登录

### 4. 消息沟通模块

- **聊天记录**：历史消息保存

#### 4.2 通知系统

## 数据库设计

### 核心数据表

#### 1. 用户相关表
```sql
-- 用户档案表
profiles (
    id UUID PRIMARY KEY REFERENCES auth.users(id),
    username VARCHAR(50),
    avatar_url VARCHAR(255),
    role ENUM('student', 'company'),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
)

-- 学生信息表
students (
    id UUID PRIMARY KEY REFERENCES profiles(id),
    student_id VARCHAR(20) UNIQUE,
    real_name VARCHAR(50),
    school VARCHAR(100),
    major VARCHAR(100),
    grade VARCHAR(20),
    phone VARCHAR(20),
    email VARCHAR(100),
    skills TEXT[],
    experience TEXT,
    resume_url VARCHAR(255),
    is_verified BOOLEAN DEFAULT FALSE
)

-- 企业信息表
companies (
    id UUID PRIMARY KEY REFERENCES profiles(id),
    company_name VARCHAR(100),
    industry VARCHAR(50),
    scale VARCHAR(20),
    address TEXT,
    contact_person VARCHAR(50),
    contact_phone VARCHAR(20),
    business_license VARCHAR(255),
    is_verified BOOLEAN DEFAULT FALSE
)
```

#### 2. 岗位相关表
```sql
-- 兼职岗位表
jobs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    company_id UUID REFERENCES companies(id),
    title VARCHAR(200) NOT NULL,
    description TEXT,
    salary_range VARCHAR(50),
    location VARCHAR(100),
    work_type VARCHAR(20),
    requirements TEXT,
    application_count INTEGER DEFAULT 0,
    status ENUM('active', 'inactive', 'filled'),
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
)

-- 岗位申请表
applications (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    job_id UUID REFERENCES jobs(id),
    student_id UUID REFERENCES students(id),
    status ENUM('pending', 'reviewing', 'accepted', 'rejected'),
    application_date TIMESTAMP DEFAULT NOW(),
    reviewed_at TIMESTAMP,
    cover_letter TEXT
)
```

#### 3. 消息相关表
```sql
-- 消息表
messages (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    sender_id UUID REFERENCES profiles(id),
    receiver_id UUID REFERENCES profiles(id),
    content TEXT,
    message_type ENUM('text', 'file', 'system'),
    file_url VARCHAR(255),
    is_read BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT NOW()
)

-- 评价表
reviews (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    reviewer_id UUID REFERENCES profiles(id),
    reviewed_id UUID REFERENCES profiles(id),
    rating INTEGER CHECK (rating >= 1 AND rating <= 5),
    comment TEXT,
    review_type ENUM('student_to_company', 'company_to_student'),
    created_at TIMESTAMP DEFAULT NOW()
)
```

### 后端API设计

#### 认证相关接口
- `POST /auth/register` - 用户注册
- `POST /auth/login` - 用户登录
- `POST /auth/logout` - 用户登出
- `POST /auth/refresh` - 刷新令牌

#### 岗位相关接口
- `GET /jobs` - 获取岗位列表
- `GET /jobs/:id` - 获取岗位详情
- `POST /jobs` - 创建岗位（企业）
- `PUT /jobs/:id` - 更新岗位
- `DELETE /jobs/:id` - 删除岗位

#### 申请相关接口
- `POST /applications` - 提交申请
- `GET /applications` - 获取申请列表
- `PUT /applications/:id` - 更新申请状态

#### 消息相关接口
- `GET /messages` - 获取消息列表
- `POST /messages` - 发送消息
- `PUT /messages/:id/read` - 标记消息已读
