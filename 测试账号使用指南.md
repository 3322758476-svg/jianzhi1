## 故障排除

### 常见错误及解决方案

**1. "Invalid login credentials"**
- 原因：账号不存在或密码错误
- 解决方案：使用上述方法创建测试账号

**2. "Email not confirmed"**
- 原因：邮箱未确认
- 解决方案：在 Supabase Dashboard 中手动确认邮箱

**3. "User not found"**
- 原因：用户资料表数据缺失
- 解决方案：运行完整的数据迁移脚本

**4. 网络连接问题**
- 检查 `.env` 文件中的 Supabase 配置
- 确认网络连接正常

### 数据库迁移检查

确保已执行完整的数据库迁移：

1. 参考 `SUPABASE_SETUP_GUIDE.md` 中的 SQL 脚本
2. 在 Supabase SQL Editor 中执行所有迁移语句
3. 确认以下表已创建：
   - `profiles`
   - `companies` 
   - `resumes`
   - `jobs`
   - `applications`
